Sedna Sample Import Form:

  • the file must be in csv format
  • This function is designed to import a csv export of herring sample data collected in Sedna
  • the order of the columns in the file is NOT important
  • the file must have the follow format:
uuid type sample_date survey_id sampler fishing_area latitude_n longitude_w gear experimental_net_used catch_weight_lbs total_fish_measured total_fish_preserved remarks creation_date last_modified_date
This is a unique universal identifier assigned to a sample from sedna. This will be stored as the old_id for historical reference This will always be stored a 2, i.e., sea sample Date of the set from which the sample originated This is the mission_number field in the sedna shared_models.Cruise object Name of sampler NAFO fishing area In this case, the starting latitude is provided in decimal degrees In this case, the starting longitude is provided in decimal degrees Gear type used (NAFO gear code) This is by default always set to yes Total weight of the catch in lbs Total number of fish measure in the originating set Total number of fish measure kept in the originating set miscellaneous remarks Date the sample record was created Date the sample record was last modified
